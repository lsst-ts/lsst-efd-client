##########
EFD Client
##########

The EFD Client provides a Python interface to query Rubin Observatory Telemetry from the Engineering and Facility Database (EFD). 
The EFD Client is built on top of the `Rubin Science Platform`_ and can be used in Jupyter notebooks or Python scripts.
